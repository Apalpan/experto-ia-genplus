La nueva fuente `Guia_Maestra_IA_Sistema_Conocimiento_2026.txt` agrega una segunda capa de conocimiento operativo. No reemplaza la documentacion anterior: la expande hacia madurez corporativa, arquitectura agentic, razonamiento avanzado, aprendizaje con evidencia y prompts aplicables a AEC.

- `AI_AGENTIC_ARCHITECTURE_FRAMEWORKS_2026.md`: arquitectura agentic, estado, herramientas, memoria, HITL, observabilidad y comparativa CrewAI/AutoGen/LangGraph.
- `AI_REASONING_TTC_GRPO_MASTERCLASS.md`: Test-Time Compute, GRPO, System 1/System 2, router cognitivo y verificadores.
- `AI_AECODE_ADAPTIVE_LEARNING_SYSTEM.md`: sistema de aprendizaje IA para AECODE, skill graph, evidencias, rubricas, tutor y Skill Passport.
- `AI_PROMPT_LIBRARY_AEC_AGENTIC.md`: biblioteca de prompts para diagnostico IA, agentes, contratos, RAG, codigo, PMO, seguridad y roadmap personal.
- `AI_ULTRA_ADVANCED_AEC_AGENTIC_USE_CASES.md`: 20 casos agentic aplicados a AEC, GEN+, PMO, BIM, obra, comercial y AECODE.
- `AI_GOVERNANCE_EVALS_OBSERVABILITY_GUIDE.md`: gobierno, evals, trazas, guardrails, aprobaciones, incidentes y calidad de produccion.
- `AI_AEC_MASTER_RESEARCH_SYNTHESIS_2026.md`: sintesis maestra de los dos reportes GEN+ / AECODE IA AEC.
- `AI_AEC_ROI_METRICS_AND_PRODUCT_WEDGES.md`: ROI, unidades economicas y cuñas producto para pilotos GEN+.
- `AI_AEC_AGENTIC_STACK_AND_PATTERNS.md`: stack agentic AEC, capas, patrones y controles.
- `AI_AEC_PROMPTS_GLOSSARY_AND_TRAINING_BANK.md`: prompts, glosario y banco de preguntas para el entrenador.

### Como estudiar esta capa

1. Empieza por madurez 100-500 para ubicar el nivel de una empresa.
2. Pasa a arquitectura agentic para entender como construir sistemas con estado y herramientas.
3. Estudia TTC/GRPO para entender por que razonamiento, costo y calidad se disenan.
4. Usa el documento AECODE para convertir aprendizaje en habilidad verificable.
5. Practica con la biblioteca de prompts hasta producir evidencias reales.
