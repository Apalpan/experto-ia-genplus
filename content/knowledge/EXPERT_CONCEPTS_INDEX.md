## Niveles

### Intermedio

El usuario puede explicar el concepto, reconocer cuándo usarlo y aplicarlo en un caso simple.

### Avanzado

El usuario puede diseñar arquitectura, datos, flujo, riesgos y criterios de aceptación.

### Experto

El usuario puede evaluar tradeoffs, seguridad, costo, calidad y escalabilidad.

### Ultra avanzado

El usuario puede diseñar sistemas agentic productivos con memoria, herramientas, evaluación, observabilidad, seguridad, permisos y mejora continua.

## Regla de estudio

No avanzar por lectura pasiva. Para cada concepto:

1. Explícalo en lenguaje simple.
2. Da un ejemplo GEN+ o AEC.
3. Define dónde falla.
4. Diseña un mini flujo.
5. Escribe una pregunta de conexión.
6. Produce una evidencia.
